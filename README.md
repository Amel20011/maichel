# Bot WhatsApp Toko Online - Liviaa Astranica

Bot WhatsApp untuk toko online dengan fitur lengkap menggunakan @whiskeysockets/baileys.

## Fitur Utama

1. **Sistem Verifikasi Pengguna**
   - Pengguna baru harus verifikasi dengan button
   - Database pengguna terintegrasi

2. **Toko Online Lengkap**
   - Daftar produk dengan button
   - Keranjang belanja
   - Checkout otomatis
   - Sistem pesanan

3. **Pembayaran QRIS**
   - Support QRIS payment
   - Multiple payment methods
   - Notifikasi ke owner

4. **Fitur Grup Lengkap**
   - Welcome message dengan video
   - Anti link, anti virtex, anti delete
   - Management member
   - Voting system

5. **Downloader Media**
   - YouTube MP3/MP4
   - TikTok, Instagram, Facebook

6. **Admin/Owner Tools**
   - Broadcast message
   - Management produk
   - Management order
   - User management
